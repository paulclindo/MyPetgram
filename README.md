
# My personal Petgram 

A Clone of Instagram for pets using Graphql, React, Webpack, Styled Components

### Prerequisites

Install

```
npm install
```

### Installing

For development env running

```
npm run dev
```
For production env

```
npm run build
```

## Running the tests

```
npm run test
```

## Built With

React, webpack, Graphql & Jest
